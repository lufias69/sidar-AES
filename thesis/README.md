## Pedoman Penulisan

- Bahasa: Indonesia
- Format akhir: akan dikonversi ke Word/PDF sesuai template kampus
- Referensi: gunakan gaya sitasi kampus (atau APA 7th saat drafting)
- Gambar/Tabel: simpan di folder `figures/` dan referensikan relatif dari file BAB
